# Datasets

## About

S4038+S2648.txt :

(1) PDB Id: The PDB entry of the protein.

(2) Mutated Chain: The chain in the 3D structure where the mutation occurs.

(3) Mutation_PDB: The mutation that occurs.

(4) UniProt: The UniProt ID of the mutated protein.

(5) Mutation_UNP: The mutation of the protein as annotated on UniProt.

(6) DDGexp: Experimentally determined ΔΔG value upon mutation.

(7) PremPS,ThermoMPNN,ThermoNet,ACDC-NN,DDGun3D,MAESTRO,MultiMutate,SimBa-SYM,FoldX,RaSP: Predictions from 10 different methods.

(8) RF-Ensemble-BalAvg: Predictions from regression model RF-Ensemble-BalAvg.

(9) RF-Ensemble-BalVote: Predictions from classification model RF-Ensemble-BalVote.

(10) Is_S4038: Indicates whether the mutation is present in the S4038 dataset.

(11) Is_S2648: Indicates whether the mutation is present in the S2648 dataset.

(12) Is_S6373: Indicates whether the mutation is present in the S6373 dataset.

(13) Is_S1030: Indicates whether the mutation is present in the S1030 dataset.
