# The example output files

## 1. Output files

(1) **example_100muts_reg_output.txt** and **example_100muts_cls_output.txt**: Regression and classification results for the 100-mutation example.

(2) **example_500muts_reg_output.txt** and **example_500muts_cls_output.txt**: Regression and classification results for the 500-mutation example.

(3) **example_1000muts_reg_output.txt** and **example_1000muts_cls_output.txt**: Regression and classification results for the 1000-mutation example.



## 2. Columns:

(1) pdb_mut: The pdb_mut column represents the mutation along with the PDB ID and chain where the mutation occurs.

(2) DDGexp: Experimentally determined ΔΔG value upon mutation.

(3) PremPS,ThermoMPNN,ThermoNet,ACDC-NN,DDGun3D,MAESTRO,MultiMutate,SimBa-SYM,FoldX,RaSP: Predictions from 10 different methods.

(4) RF-Ensemble-BalAvg: Regression predictions generated by the RF-Ensemble-BalAvg model for each mutation.

(5) RF-Ensemble-BalVote: Classification predictions generated by the RF-Ensemble-BalVote model for each mutation.
