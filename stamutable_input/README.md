# The example input files

(1) pdb_mut: The pdb_mut column represents the mutation along with the PDB ID and chain where the mutation occurs.

(2) DDGexp: Experimentally determined ΔΔG value upon mutation.

(3) PremPS,ThermoMPNN,ThermoNet,ACDC-NN,DDGun3D,MAESTRO,MultiMutate,SimBa-SYM,FoldX,RaSP: Predictions from 10 different methods.
